Understanding NoAuth

What is NoAuth?

NoAuth is a trust-based authorization mechanism, but it is also much more than that. NoAuth is an initiative to re-introduce trust into the wider internet.

Where traditional security frameworks require convoluted and often misguided metaphors for trust like handshakes and keys, NoAuth understands that even between APIs, real friends have no secrets. All that is required in order to get started with NoAuth is the mutual understanding of trust between two participating parties.

NoAuth Version 1

The first implementation of the NoAuth specification. Contains some vulnerabilities addressed in NoAuth2. While still a valid security mechanism, those interested in encouraging good behaviors in clients should use NoAuth2

NoAuth Version 2
